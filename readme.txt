== DESCRIPTION ==
Supports a single shortcode with parameters

[knowledgebase subcats=true|false columns=[1-9] excludes=[CSV] includes=[CSV]]

subcats - should it read in posts only at top category level or go deeper
columns - currently vertical only supports one
excludes - categories to ignore, comma separated values
includes - only includes these categories, comma separated values

Based on implementation from Guido07111975 for VSKB
